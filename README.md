📊 Organizador de Declaração de Imposto de Renda
Autor: Amanda Borges
Sistema: LION APP
Plataforma: Microsoft Excel
Finalidade: Organização prévia de informações para Declaração de Imposto de Renda Pessoa Física (IRPF)

🧾 Visão Geral
O Organizador de Declaração de Imposto de Renda é uma planilha desenvolvida para facilitar a organização, centralização e conferência de dados necessários para o preenchimento da Declaração de Imposto de Renda Pessoa Física.
O arquivo permite reunir, de forma estruturada, os dados do titular, informes de rendimentos bancários e entradas financeiras mensais, reduzindo erros, esquecimentos e retrabalho no momento da declaração.

⚠️ Este arquivo não substitui o programa oficial da Receita Federal.
Ele serve como ferramenta de apoio e organização.


🧭 Estrutura do Arquivo
O organizador é dividido em 3 abas principais, com navegação facilitada por botões.

1️⃣ TITULAR – Dados do Contribuinte
Nesta aba devem ser preenchidas as informações pessoais do titular da declaração.
Campos disponíveis:

Nome completo
CPF
Data de nascimento
Título de eleitor
Cônjuge
Endereço completo (rua, número, bairro, CEP)
Telefone
Celular
E-mail
Informações complementares:

Houve alterações em relação ao ano anterior
Dependente cônjuge
Residente no exterior



📌 Objetivo: Centralizar os dados cadastrais para facilitar conferência e preenchimento no programa da Receita Federal.

2️⃣ INFORMES – Rendimentos Bancários
Nesta aba são registrados os informes de rendimentos fornecidos pelos bancos e instituições financeiras.
Funcionalidades:

Registro de múltiplos bancos
Campo para:

Nome do banco
Código do banco
Valor total informado
Anexo do informe (PDF ou arquivo digital)


Cálculo automático do valor total de rendimentos bancários

📌 Objetivo: Garantir que todos os informes bancários sejam declarados corretamente e evitar omissões.

3️⃣ NOTAS – Entradas Financeiras / Holerites
Esta aba é destinada ao controle de entradas mensais de receita, como salários, holerites ou outros rendimentos.
Campos da tabela:

Data (mês/ano)
Categoria (ex.: Holerite, Receita Extra, Outros)
Valor

📌 Objetivo: Facilitar a conferência de rendimentos ao longo do ano e cruzar informações com informes bancários e declaração final.

🔀 Navegação
O arquivo possui botões de navegação:

PRÓXIMO →
← ANTERIOR

Esses botões permitem avançar e retornar entre as abas de forma intuitiva, sem necessidade de usar as abas inferiores do Excel.

✅ Boas Práticas de Uso

Preencha os dados com atenção e conferência documental
Anexe sempre que possível os informes oficiais em PDF
Utilize valores exatamente como constam nos informes
Atualize o arquivo sempre que receber novos documentos
Guarde o arquivo em local seguro, pois contém dados sensíveis


🔒 Segurança e Privacidade
Este arquivo contém informações pessoais e financeiras.
Recomenda-se:

Uso de senha no arquivo Excel
Backup em local seguro
Não compartilhar sem necessidade
