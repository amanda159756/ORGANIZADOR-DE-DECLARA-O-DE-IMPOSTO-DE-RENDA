# 📊 Organizador de Declaração de Imposto de Renda (Excel)

Organizador em Excel para facilitar a **coleta, organização e conferência** das informações necessárias para a **Declaração de Imposto de Renda Pessoa Física (IRPF)**.

> Este projeto tem como objetivo **organizar dados**, não substituindo o programa oficial da Receita Federal.

---

## 🧾 Visão Geral

O **Organizador de Declaração de Imposto de Renda** foi desenvolvido para centralizar, em um único arquivo, os principais dados exigidos na declaração anual, reduzindo erros, esquecimentos e retrabalho.

Ideal para:
- Uso pessoal
- Organização prévia antes de enviar ao contador
- Conferência de informes e rendimentos

---

## 🗂️ Estrutura do Arquivo

O arquivo Excel é dividido em **3 abas principais**, com navegação por botões:

### 1️⃣ Titular
Cadastro completo do contribuinte:
- Nome
- CPF
- Data de nascimento
- Título de eleitor
- Dados do cônjuge
- Endereço
- Telefones
- E-mail
- Informações complementares (residência no exterior, alterações do ano anterior, dependentes)

---

### 2️⃣ Informes
Registro dos **informes de rendimentos bancários**:
- Banco
- Código do banco
- Valor informado
- Anexo do informe (PDF)
- Cálculo automático do valor total

---

### 3️⃣ Notas
Controle de **entradas financeiras mensais**:
- Data (mês/ano)
- Categoria (ex: holerite, receita extra)
- Valor

---

## 🔀 Navegação

O arquivo conta com botões:
- **PRÓXIMO →**
- **← ANTERIOR**

Facilitando a navegação sem necessidade de usar as abas do Excel.

---

## ✅ Como Usar

1. Baixe o arquivo Excel
2. Preencha as abas na ordem:
   - **Titular**
   - **Informes**
   - **Notas**
3. Anexe os informes bancários quando possível
4. Utilize os dados organizados para preencher o programa da Receita Federal

---

## 🔒 Segurança

Este arquivo contém **dados pessoais e financeiros**.  
Recomenda-se:
- Proteger o arquivo com senha
- Armazenar em local seguro
- Não compartilhar informações sensíveis
